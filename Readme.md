# Backend Project
A scalable backend system that combines core features of a video-sharing platform and microblogging system. Built using Node.js, Express, and MongoDB, this project supports video management, tweets, likes, comments, subscriptions, and channel analytics.

---

## 📌 Introduction

This backend application integrates functionalities similar to YouTube and Twitter, providing a unified system for content creation, interaction, and analytics. It follows a modular architecture with optimized database queries and scalable API design.

---

## 🚀 Features

### 🔐 User Management

* User registration, login, logout, and password management
* Profile management (avatar, cover image, details)
* Watch history tracking

### 📹 Video Management

* Video upload and publishing
* Search, sorting, and pagination
* Video editing and deletion
* Visibility control (publish/unpublish)

### 🐦 Tweet Management

* Create and publish tweets
* View user tweets
* Update and delete tweets

### 🔔 Subscription Management

* Subscribe and unsubscribe to channels
* View subscriber and subscribed channel lists

### 📂 Playlist Management

* Create, update, and delete playlists
* Add/remove videos from playlists
* View user playlists

### 👍 Like Management

* Like and unlike videos, comments, and tweets
* Retrieve liked videos

### 💬 Comment Management

* Add, update, and delete comments on videos

### 📊 Dashboard

* View channel statistics (views, subscribers, videos, likes)
* Access uploaded videos

## 🛠️ Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* dotenv

## 📌 Core Concepts

* RESTful API design
* MVC architecture
* MongoDB aggregation pipelines
* Pagination using skip and limit
* Centralized error handling
